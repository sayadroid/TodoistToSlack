# TodoistToSlack
TodoistのタスクをSlackに投げるアプリ

## Usage
1. config/config.yaml.template をコピーした、config/config.yaml を作成し、Todoist,Slackの各情報を記入
2. プロジェクトルートで ruby main.rb を実行
